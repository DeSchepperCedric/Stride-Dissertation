The Stride project.

The acronym Stride stands for Simulation of transmission of infectious diseases.

Documentation

The the documentation for the current version of the Stride project consists of:

  • Reference documentation generated from the source using the doxygen tool.
  • User manual generated from latex sources and available in pdf format.
  • Installation information is provided in a INSTALL.txt file in the distribution.
  • The file PLATFORMS.txt lists platforms on which the code was built and tested.
  • The file DEPENDENCIES.txt lists the softwares that the project depends on.

Following the installation of the project software, the Reference documentation and User Manual are available in the doc subdirectory of the installation directory. The other files are available in the root directory of the project software.
