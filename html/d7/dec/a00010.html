Detailed Description

Benchmark test descriptor.

Benchmark tests descriptors are specified by

  • the group the belong to in combination with their individual name. This combination is referred to as the canonical name.
  • the number of times the benchmark has to be executed
  • the factory that produces test instances through its call operator
  • the factory that produces info about the test (intended to be useful for viewers)
  • a status variable that lets you (temporarily) disable a tests so it remains registered but wil not run when the benchmarks execute.
